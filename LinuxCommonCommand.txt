работа с терминалом:

Hot Key:
ctrl + l - очистить терминал
ctrl + c - остановить выполнение

Команды:
clear - очистить терминал
&& - последовательное выполнение команд;
	пример -
	(cd test_dir/ && touch fileName1.txt - 
		перейти в директорию test_dir/ и создать файл fileName1.txt)

Работа с директориями:

cd - перейти в домашний каталог
cd [path] - перейти в директорию по пути path(путь регистрозависимый)
cd .. - подняться на уровень выше

mkdir [pathDir] - make directory, создание директории
rmdir [pathDir] - remove directory, удалить директорию, но только пустую

Работа с файлами:
touch [fileName1] [fileName2] - создать файл или несколько файлов с именем и расширением
> [fileName] - создать файл с именем и расширением

cat [pathFile1] [pathFile2] - сокращение от concatinate, вывести содержимое файла, или нескольких фалов

rm [pathFile] - удалить файл
rm [fileName*] - удалить файлы, у которых начало имени одинаковое
rm -r [pathDir]- рекурсивное удаление фалов в каталоге и вложенных каталогов

cp [filePath1] [filePath2] - сокращение от copy, скопировать файл из filePath1 в filePath2
cp -r [pathDir1] [pathDir2]- рекурсивно скопировать каталог с файлами из pathDir1 в pathDir2

mv [filePath1] [filePath2] - сокращение от move, переместить файл. 
	Если переместить файл в ту же директорию, 
		но сдругим именем, то файл просто переименуется


Команды по apt (advanced packages tool): 
	apt это более ранняя утилита, для работы с пакетами
	apt-get - то же самое, но появившаяся чуть позже
	у apt есть свой список репозиториев и списки пакетов в каждом репозитории
	
man apt - это help для apt

apt update - пройдёт по всем репозиториям и обновит для каждого списки пакетов
apt list --upgradable - получить список пакетов
apt upgrade -y - обновить установленные пакеты до новых актуальных версий в репозиториях
				 флаг -y будет подтверждать соглащение на обновление
apt full-upgrade -y - обновить установленные пакеты до новых актуальных версий в репозиториях 
						+ установит все необходимые для этого зависимые пакеты и удалит старые,
						флаг -y будет подтверждать соглащение на обновление
apt install [packageName1] [packageName2] - установить пакет или несколько 
						(если не указать версию, то будет установлен последний)
apt install [packageName1] [packageName2] + двойной tab - выпадет список с возможными пакетами для установки 
apt install [packageName1=version] - установить пакет с version (знак = пишется без пробелов)
apt show [packageName1] - показать информацию о пакете
apt search [packageName1] - поиск пакета packageName1 как доступного в repo так и установленного
apt remove -y [packageName1] - удаление пакета, без удаления файлов конфигурации (-y для согласия)
apt purge -y [packageName1] - удаление пакета с файлами конфигурации (-y для согласия)
apt autoremove - удаление зависимых пакетов, но кроме тех, которые нужны другим пакетам
apt clean - удалит пакеты скачанные для установки с локальных дисков
apt autoclean - удаляет дистрибутивы пакетов и кеш файлы
